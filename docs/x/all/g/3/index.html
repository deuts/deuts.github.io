In n8n, “Bearer Auth” and “Header Auth” are functionally identical if you set the header to Authorization: Bearer . Pick Bearer Auth when the server expects that header — it’s self-documenting. Pick Header Auth for non-standard headers like X-API-Key. That’s the only difference. #n8n #selfhosted
